## 🌟 Executive Overview

**ATSResume_Creation** is a modern intelligent resume builder designed to create **ATS-compliant resumes** that pass Applicant Tracking Systems and attract recruiters.

🎯 Built for:

✅ Freshers
✅ Experienced Professionals
✅ Career Switchers
✅ Job Seekers
✅ Recruiters / HR Teams

This platform helps users generate optimized resumes faster with modern formatting and keyword-rich content.

# 🚀 How to Run

## 1️⃣ Clone Repository

```bash id="v7n3ea"
git clone https://github.com/Vinothkumar-SV/ATSResume_Creation.git
cd ATSResume_Creation
```

## 2️⃣ Install Dependencies

```bash id="m2q6uf"
npm install
```

## 3️⃣ Start Application

```bash id="c5k1ry"
npm run dev
```

## 4️⃣ Open Browser

```text id="n8p4dw"
http://localhost:5173
```
